# Esiasennus-apuri(ni)

Itselleni hyödyllisiä komentoja Windowsia asentaessa, käytin oppisopimuspaikassa hyödyksi kaikkia näitä, skriptan tein vasta puolivälissä ja harmittaa etten ollut aikaisemmin tehnyt.

## Käyttö

Avaa __GUI.ps1__ tai aja suoriltaan kaikki komennot kerralla avaamalla __AIO.ps1__ järjestelmänvalvojana. Testattu PowerShell 7.

## Komennot GUI:ssa, tai aja kaikki avaamalla AIO.ps1 
* Asentaa suomenkielisen kielipaketin Windows 10&11 (testattu Win11) 

* Alueenvaihto ja aikaformaatti suomeen

* Aikapalvelimeksi mikes

* Tehtäväpalkin käynnistä-painike vasemmalle puolelle

* Muuttaa läppärissä virta-asetukset niin, ettei kone mene lepotilaan kannen sulkiessa

* Pakottaa ryhmäkäytännöt voimaan


## MUUTA
Skriptat löydät kansiosta __/skriptit/__, jotka voit lukea millä tahansa tekstieditorilla. 
Lisäilen todennäköisesti uusia skriptejä, kun tarve tulee. pohja 5/5, tuskin tulen GUI päivittämään.
